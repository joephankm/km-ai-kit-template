# Skills

A catalog of all skills in this library. Use this as a reference when looking for a skill to activate in your
AI tool. Each skill is a slash command or instruction set that can be loaded into your AI tool's commands directory.

Skills are stored under `skills/<category>/` and can be a single `.md` file (flat) or a folder with a `SKILL.md`
entry file.

> **This is a template.** The categories and entries below are examples only — they don't exist in the repo yet.
> Replace them with your real skills as you build up your library.
